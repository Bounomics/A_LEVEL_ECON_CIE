# A-Level Economics CIE (9708) Question Papers Index

This index provides direct download links to all Economics A-Level question papers organized by examination session.

**Naming Convention:**
- `S##` = Summer (May/June) session
- `W##` = Winter (October/November) session
- `##` = Year (21 = 2021, 22 = 2022, etc.)
- `qp_XY` = Question Paper (X = paper number, Y = variant)
